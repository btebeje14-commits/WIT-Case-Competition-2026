# WIT-Case-Competition-2026
📌 CareMetrics: Pay for Value, Not Volume
A Value-Based Medicaid Rate-Setting Intelligence System
🧠 Problem Statement

State Medicaid programs rely heavily on claims and financial data to set capitation rates for Managed Care Organizations (MCOs). However, this approach is often:

Retrospective (lagging indicators)
Limited in clinical and behavioral insight
Lacking transparency for stakeholders

As a result, rates may not accurately reflect true healthcare value or efficiency.

💡 Our Solution

CareMetrics is a technology-enabled value-based care intelligence system that enhances Medicaid rate-setting by integrating:

Financial data (claims, capitation rates)
Clinical data (diagnoses, service usage)
Patient engagement data (surveys, adherence, feedback)

It transforms raw healthcare data into:

Efficiency scores, performance insights, and transparent value-based rate adjustments.

🏗️ System Architecture

Our system follows a 4-step pipeline:

Ingest & Validate
Secure data ingestion
Data quality checks
Standardization across sources
Analyze & Score
Efficiency modeling
Value-Based Purchasing (VBP) scoring
Risk-adjusted performance analysis
Monitor & Alert
Real-time performance tracking
Anomaly detection
Early warning indicators
Display & Report
State dashboards
MCO performance dashboards
Patient-level engagement insights
📊 Key Features
Integrated Medicaid data platform
Patient engagement tracking system
Efficiency scoring for MCOs
Value-based rate adjustment support
Transparent reporting dashboards for stakeholders
👥 Stakeholders
State Governments → rate-setting & oversight
MCOs / Health Plans → performance optimization
Mercer Consultants → analytics & policy modeling
Providers → care improvement insights
Patients → engagement & health tracking
📈 Impact

CareMetrics improves Medicaid systems by:

Increasing transparency in rate-setting
Improving accuracy of capitation rates
Aligning payments with patient outcomes
Reducing inefficiencies in care delivery
